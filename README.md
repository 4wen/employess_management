# employess_management
这是在学习SSM时写的练手Demo。

#### 功能点

+ 分页
+ 数据校验
  + jquery前端校验+JSR303后端校验

+ ajax
+ Rest风格URI

#### 技术点

+ 基础框架-ssm（SpringMVC+Spring+MyBatis）
+ 前端框架-bootstrap
+ 项目的依赖管理-Maven
+ 分页-pagehelper
+ 逆向工程-MyBatis Generator
